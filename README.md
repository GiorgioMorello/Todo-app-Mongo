# Todo app Mongo
 Um Todo app feito com Flask e MongoDB



# 1º
### Ao entrar no site, você pode escrever alguma tarefa no campo e clicar em "ADD" para criar um item e armazenar no banco de dados

# 3º 
### O botão "Not completed" qunado pressionado vai alterar o texto voltando para a forma padrão e ira alterar o campo "complete" para False no banco de dados


# 4º 
### Botão "Delete Completed" vai deletar todos os itens que estão com o campo "complete" igual a True
